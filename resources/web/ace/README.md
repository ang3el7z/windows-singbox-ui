# Ace Editor

Эта папка содержит локальные файлы Ace Editor для использования в приложении.

## Файлы

### Обязательные:
- `ace.js` - Основной файл Ace Editor (ядро редактора)
- `mode-json5.js` - Режим подсветки синтаксиса для JSON5
- `theme-monokai.js` - Тема Monokai для редактора

### Дополнительные (рекомендуемые):
- `worker-json.js` - Воркер для валидации JSON в реальном времени (подчеркивает ошибки)
- `ext-language_tools.js` - Расширение для автодополнения и подсказок

## Обновление

Для обновления Ace Editor запустите скрипт:

```bash
py scripts/download_ace.py
```

Или вручную скачайте файлы с https://cdn.jsdelivr.net/npm/ace-builds@1.23.4/src-min-noconflict/

## Версия

Текущая версия: 1.23.4

## Использование

Все файлы автоматически включаются в сборку приложения через `SingBox-UI.spec`.
При запуске приложения файлы загружаются локально, без необходимости интернет-соединения.
